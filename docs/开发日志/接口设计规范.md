
# 接口设计规范

## http响应码

| code | 含义          | 使用场景        |
| ---- | ----------- | ----------- |
| 200  | 请求成功        | 查询、修改       |
| 201  | 创建成功        | 注册、新增       |
| 400  | 参数错误        | 表单问题        |
| 401  | 未登录/token失效 | JWT         |
| 403  | 权限不足        | RBAC        |
| 404  | 数据不存在       | 用户不存在       |
| 409  | 数据冲突        | 手机号重复       |
| 500  | 服务器错误       | try catch异常 |

## 返回数据格式

```js
{
  code: 200,
  success: true,
  message: "操作成功",
  data: {}
}
```
